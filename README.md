# 🚜 Bulldozer Price Predictor

A Streamlit web app that predicts bulldozer resale prices using a trained Random Forest model.

## ✨ Features

* Predict bulldozer sale price instantly
* Clean and modern UI
* Inputs: Year Made, Machine Hours, Sale Date
* Built with Streamlit + Machine Learning

## 📦 Requirements

```bash
pip install streamlit pandas joblib scikit-learn
```

## ▶️ Run App

```bash
streamlit run app.py
```
